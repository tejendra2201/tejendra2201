Create a script that asks for username and it tells you the id of the user and tells if id is greater 1000 or not?

read -p "Enter the username: " usr
uid=$(cat /etc/passwd | grep "$usr" | cut -d':' -f3)

if [[  $uid > 1000 ]]
	then
		echo "UID is greater than 1000"
	else
		echo "UID is less than 1000"
fi


Create a script that asks for username and tells u if it exists on the system or not. If exists, then display the shell user

read -p "Enter the username: " usr

userinfo=$(cat /etc/passwd | grep "$usr")

if [[ -n $userinfo ]] //if the variable userinfo is non zero
	then 
		echo "user exists"
		shell=$(cat /etc/passwd | grep "$usr" | cut -d':' -f7)
		echo "Shell being used by $usr is : $shell"
	else
		echo "user does not exist"
fi

Package installations in Linux
------------------------------
Ques - Differentiate between rpm and yum

Ques - if yum is installing some package, which process runs in the background for the same?

Installation of package using rpm
---------------------------------
rpm -ivh <packagename>.rpm
i - install
v - verbose
h - print hashes to show that download is gong on
Sample: ################################################# 100%

Command to remove a package
rpm -e <packagename>.rpm
Command to update a package
rpm -U <packagename>.rpm

To check if a package is installed or not, we used
rpm -qa | grep <packagename>
--
Example:
[root@localhost localrepo]# rpm -qa | grep http

Installation of package using yum
---------------------------------
yum -y install <packagename>
yum update
yum upgrade
yum remove <packagename>
yum list   //is  used  to  list  various  information about available packages

To copy packages on the machine from ISO
----------------------------------------
Step1 - Connect the iso from Settings of the machine
Step2 - Mount the iso -> mount -t iso9660 /dev/sr0 /mnt
Step3 - Create a folder and Copy media content to that specific folder
mkdir /localrepo
cp -rv /mnt/* /localrepo/
Step4 - Create the repo
Note: before repo creation , move the default repos to some other location
#mkdir ../test
#mv * ../test/

[root@localhost localrepo]# cat /etc/yum.repos.d/local.repo 
[centos7]
name=centos7
baseurl=file:///localrepo/
enabled=1
gpgcheck=0

Step5 - createrepo /localrepo
Step6 - yum clean all
Step7 - yum list; yum repolist all

To make it work via httpd
-------------------------
Modify the repo - use http://
[root@localhost conf.d]# cat /etc/yum.repos.d/local.repo 
[centos7]
name=centos7
baseurl=http:///localrepo/
enabled=1
gpgcheck=0
